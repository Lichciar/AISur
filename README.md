# AISur
AI trying to survive after the apocalypse.

Описание:
	Искуственный интелект на основе нейронной сети будет пытаться выжить в придуманых нами условиях.

Язык программирования:
	СИ++.

Сборка:
	Запустить скрипт install.sh.
